# English
# 📚 BookStore

**BookStore** — a full-featured CRUD application for managing a book catalog. Built with a modern tech stack using clean architecture and containerization.

## 🚀 Tech Stack

### Backend
- **ASP.NET Core 10** — API server
- **Entity Framework Core 10** — ORM for database operations
- **PostgreSQL** — relational database
- **Swagger / Swashbuckle** — API documentation
- **Docker & Docker Compose** — containerization

### Frontend
- **React 19** — user interface
- **Axios** — HTTP requests to API

## 📋 Features

- ✅ **Create** — add new books
- 📖 **Read** — view book list and details
- ✏️ **Update** — edit book information
- 🗑️ **Delete** — remove books
- 📑 **Swagger UI** — interactive API documentation

# Русский

# 📚 BookStore

**Книжный магазин** — полноценное CRUD-приложение для управления каталогом книг. Реализовано на современном стеке технологий с использованием чистой архитектуры и контейнеризации.

## 🚀 Технологии

### Backend
- **ASP.NET Core 10** — API сервер
- **Entity Framework Core 10** — ORM для работы с БД
- **PostgreSQL** — реляционная база данных
- **Swagger / Swashbuckle** — документация API
- **Docker & Docker Compose** — контейнеризация

### Frontend
- **React 19** — пользовательский интерфейс
- **Axios** — HTTP-запросы к API

- ✅ **Create** — добавление новых книг
